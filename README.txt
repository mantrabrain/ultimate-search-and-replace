== Description ==
Search and replace text from WordPress Database. This plugins allows you to search any string (text) from databsae and replace it with your new text. Mainly while migrating your site this tool helps you to replace text via this plugin.

**Want to contribute?**

Feel free to open an issue or submit a pull request on [GitHub](https://github.com/mantrabrain/ultimate-search-and-replace).

== Installation ==

1. Install Cookie Notice either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Cookie Notice settings and set your options.

== Frequently Asked Questions ==
= Using =

Once this plugin  activated, Ultimate Search Replace will add a page under the "Tools" menu page in your WordPress admin.

= Is this plugin compatible with my hosting provider ? =
Yes! This plugin should be compatible with any of the hosting company or provider.

= What is the risk factor of this plugin? =
While doing search and replace if you enter a wrong search or replace string could damage your database. Because of this, it is always adviseable to have a backup of your database before using this plugin.
